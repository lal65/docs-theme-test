## Summary
This module was originally written to centralize various custom entity
functionality. Since then, Drupal core has added generic forms and handlers
that renders most of this module redundant. There is also much less complexity
involved with custom entities on this application than there was in ages past.

### Technical debt
This module should be eliminated, replaced with core-provided forms and
handlers.

## Requirements
This module requires no modules outside drupal core.

## Installation
Install as you would normally install a contributed Drupal module. For further information, see [Installing Drupal Modules](https://www.drupal.org/extending-drupal/installing-drupal-modules).

## Configuration
This module does not expose any configuration.
