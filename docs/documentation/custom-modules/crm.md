## Summary
The CRM Integration module provides a highly secure, defensive solution for
interfacing with the Penn State ECRM Group. It is a very strange solution for
a very unfortunate relationship with the central groups and their vendor.

## Solution Architecture

> The Applications Developers were **_NOT_** consulted about any of the overall
solution architecture, nor did they have any meaningful influence over how the
central group or vendor set anything up. This is **_NOT_** an architecture that
would have been chosen from the Applications Developers side. It was forced on
us.
