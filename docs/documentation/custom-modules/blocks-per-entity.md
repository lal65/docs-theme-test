## Summary
The Blocks per entity module adds a mechanism that allows any fieldable
entity type to be able to toggle theme level blocks on and off on a per-entity
basis. This module is deprecated.

### Technical debt
There are two plugins provided by this module that are not functionally used,
but are still referenced by fields. The fields referencing these plugins should
be deleted and that will make this module truly obsolete.

## Requirements
This module requires no modules outside drupal core.

## Installation
Install as you would normally install a contributed Drupal module. For further information, see [Installing Drupal Modules](https://www.drupal.org/extending-drupal/installing-drupal-modules).

## Configuration
This module has no configuration.
